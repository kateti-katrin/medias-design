# СМЫСЛ

СМЫСЛ — медиа-сервис о дизайне для тех, кто его заказывает. Мы пишем для владельцев бизнеса и менеджеров, которые работают с дизайнерами и хотят понимать за что платят.

## Стек

- Ruby
- Rails
- SQLite3
- ERB
- SCSS
- JavaScript

## Что сделано

- MVC-структура Rails для разделения маршрутов, логики, данных и представлений.
- Серверный рендер страниц через ERB для динамического вывода контента из базы данных.
- Модели `Article`, `Tag`, `Like`, `Comment`, `Service`, `User` для хранения контента, интерактивности и пользовательских данных.
- `slug` и `public_uuid` у статей для человекочитаемых URL и стабильных публичных идентификаторов.
- API на `/api` для лайков и комментариев без перезагрузки страницы.
- Обработка JSON-ответов и ошибок API для предсказуемой работы клиентской части.
- Счетчик просмотров статьи для фиксации пользовательской активности.
- Фильтрация статей по тегам для удобной навигации по материалам.
- Генератор ТЗ для формирования черновика задачи дизайнеру.
- Калькулятор бюджета для предварительной оценки стоимости проекта.
- Авторизация через Devise для пользовательской части приложения.
- Кастомная страница 404 для цельного пользовательского сценария.
